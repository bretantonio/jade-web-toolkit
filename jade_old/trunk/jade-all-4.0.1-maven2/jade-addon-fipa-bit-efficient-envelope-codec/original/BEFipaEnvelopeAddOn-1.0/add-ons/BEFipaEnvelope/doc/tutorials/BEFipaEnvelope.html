About Bit-efficient Envelope Codec for JADE

Author: Heikki Helin (TeliaSonera), Mikko Laukkanen (TeliaSonera), Ahti Syreeni (TeliaSonera)

Date: December 17, 2007

Java platform: Sun JDK 1.5

JADE version 3.5

If MIDP version is compiled: Java Wireless Toolkit 2.2 and LEAP add-on for JADE 3.5

JADE can use different Message Transport Protocols. The MTP HTTP (version implemented in CASCOM project) uses bit-efficient encoding of message envelope (FIPA 00088) as default. With this bit-efficient envelope-codec and the MTP HTTP users of JADE can send and receive messages with bit-efficient envelopes. Escpecially, bit-efficient envelope coding and the MTP is the way for JADE to communicate with mobile CASCOM Agent Platform. Note that this component is not directly used by JADE, the MTP HTTP implementation uses it..

Configuration, Usage and Examples

This component is used by MTPs, not directly by JADE. Please see the manual of your MTP. Please notice that this codec is currently only supported by the HTTP MTP implementation (from CASCOM Project).
